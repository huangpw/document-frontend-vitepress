## 官方文档

Bootstrap仍然是最流行的 CSS 框架之一。他们提供了大量的文档、示例和演示，可以帮你快速进行响应式 Web 开发。该框架的当前版本是 Bootstrap 5，相较于V4，做出了一些改变：

- 默认语言：English
- 官方文档：[Bootstrap](https://getbootstrap.com/)

特点：

- 拥有了自己的SVG图标库
- 不再支持jquery
- 不再支持IE
- 增加了css自定义属性
- 增强的网格系统
- Forms可被更新
