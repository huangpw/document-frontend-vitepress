# Hamburgers汉堡

## Tasty CSS-animated hamburgers-汉堡

在构建移动响应应用程序时，汉堡图标已成为一种广泛使用的模式。它通常放置在图形界面的顶角。它的目标是切换导航菜单。

- 默认语言：English
- 官方文档：https://jonsuh.com/hamburgers/

特色：

- 动画组件
- 项目在 GitHub 上可用
- 支持 95.43% 的浏览器
- ARIA 友好
- 高度可定制
- 可用的 Sass 文件

- 下载 CSS文件 并将其放在 `<head>` 标记中

- 添加标记
- 使用 css 类 hamburger--collapse 来切换状态
